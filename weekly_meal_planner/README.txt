Brief description of the project:
Weekly Meal Planner & Smart Shopping List is a desktop application built in Python for managing recipes and organising meals across a seven-day schedule. The application uses a recipe library with the ability to load and save CSV files. Furthermore, users can create, modify or delete recipes containing the following information: recipe name, category, preparation time, number of servings, calories per serving and a list of ingredient names.

A user can search recipes by name, category or ingredient using a linear search, apply a custom insertion sort, and perform an exact-name search using a recursive binary search. The weekly planner consists of breakfast, lunch and dinner slots for each day of the week, allows slots to remain empty, saves selections to a CSV file, and provides calorie estimates on a daily and weekly basis.

The shopping-list feature collects ingredients from planned meals, eliminates duplicates and saves the result as a text file. Input validation and file recovery handle common problems, including empty fields, invalid numerical input, missing files and malformed CSV rows.
